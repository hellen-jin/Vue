GitHub clone方法    
//终端 选择下载的路径  
cd ~/Desktop  
//github 复制SSH  
//终端 输入clone命令  
git clone git@github:用户名/路径.git  
//成功 该路径下可看到文件夹  
  
    
GitHub 上传方法  
//打开文件所处路径  
git init  
//把文件添加到仓库  
git add 文件名  
//把文件提交到仓库  
git commit -m "注释"  
//将文件push到已有的仓库上  
git remote add origin git@github.com:用户名/路径.git  
git push -u origin main
